# Gerador de Relatorio de Notas

Aplicativo desktop em React + Tauri para leitura de XMLs fiscais e geracao de relatorios em Excel.

## Stack

- React
- Tauri
- Rust
- TypeScript

## O que faz

- importa arquivos `.xml` e `.zip`
- identifica documentos fiscais suportados
- separa entradas, saidas e notas sem CNPJ identificado
- gera um unico arquivo Excel com abas separadas
